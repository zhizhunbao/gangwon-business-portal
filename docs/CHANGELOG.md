# 变更日志 (CHANGELOG)

本文件记录江原道企业门户项目的所有重大更新与迭代。遵循 **`dev-project_docs`** 简单性原则。

## [1.1.2] - 2026-01-25

### 🏗️ 文档架构重构

- **精简目录**: 将 `docs/` 目录从复杂的 01-10 编号结构重构为扁平化的 `requirements`, `design`, `guides`, `archive` 四大核心板块。
- **清理 Skill**: 移除了 120+ 个非软件开发相关的自动化规则（Skill），确保 AI 助手专注于编码业务。

### ✨ 新功能需求与设计

- **统计报告 (PRD-006)**: 基于客户原型生成了多维度企业统计需求文档，涵盖主导产业 KSIC 代码映射逻辑。
- **统计 API 设计**: 完成了 `/api/v1/admin/statistics/report` 接口契约设计，支持 Excel 异步导出。
- **后端功能实现与测试**: 完成了统计模块的基础开发，包括 Pydantic 参数校验、多维度筛选逻辑以及与 Supabase 的集成，并通过了自动化测试。
