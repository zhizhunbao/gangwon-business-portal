# Project Management Skill

## 描述
这个技能专门用于江原道创业企业绩效管理门户的项目管理工作，包括项目规划、进度跟踪、文档管理、团队协作等。

## 项目管理特点
- **敏捷开发**: Sprint 规划和回顾
- **任务管理**: Issue 跟踪和分配
- **进度监控**: 里程碑和交付物管理
- **文档管理**: 技术文档和用户文档
- **团队协作**: 代码审查和知识分享
- **质量保证**: 测试覆盖率和性能监控

## 使用场景
- 制定开发计划和里程碑
- 创建和管理项目任务
- 生成项目进度报告
- 进行代码审查和质量检查
- 编写技术文档和用户手册
- 团队会议和回顾总结
- 风险评估和问题解决

## 项目结构管理
```
docs/
├── project/              # 项目管理文档
│   ├── planning/         # 项目规划
│   ├── meetings/         # 会议记录
│   ├── reports/          # 进度报告
│   └── retrospectives/   # 回顾总结
├── requirements/         # 需求文档
├── architecture/         # 架构设计
├── guides/              # 开发指南
└── api/                 # API 文档
```

## 开发流程
1. **需求分析** → 功能规格书
2. **技术设计** → 架构设计文档
3. **开发实现** → 代码审查
4. **测试验证** → 质量报告
5. **部署发布** → 发布说明
6. **运维监控** → 性能报告
